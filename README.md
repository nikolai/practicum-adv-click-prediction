# Модель-классификатор для предсказания вероятности клика на рекламу

## Описание

Проект содержит исследование и построение оптимальной модели для предсказания вероятности клика пользователя по рекламе.
Ключевые слова: SVM, отбор признаков, калибровка вероятностей


## Требования

- **Python**: 3.11 или выше
- **ОС**: Linux, macOS или Windows (с WSL)

## Установка и запуск

#### Для Fish shell:

```fish
fish start.fish
```

#### Для Bash/Zsh:

```bash
./start.sh
```
